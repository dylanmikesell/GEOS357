What is MATLAB?

MATLAB ("MATrix LABoratory") is a tool for numerical computation and visualization. The basic data element is a matrix, so if you need a program that manipulates array-based data (i.e. matrix data) it is generally fast to write and run in MATLAB. If you have very large arrays or lots of computations, in which case you're better off using C or Fortran. Those languages are faster because they are compiled languages, whereas languages like MATLAB and Python are called interpreted languages. If you want to know more about the difference between compiled and interpreted languages check out these articles.
  • Programming Concepts: Compiled and Interpreted Languages
  • What’s the difference between compiled and interpreted languages?

Variables:

MATLAB is a powerful computational tool because you can write computer code and quickly debug as you go. You can quickly debug because MATLAB keeps variables loaded into memory so you can see the data. You can see which variables are loaded into memory in the Workspace pane.
You can double click any of these variables and they will open in a spreadsheet. Make sure to right-click on the top row in workspace to turn on the 'size' column.
Notice that the value and the size of the variable are shown. The size is a very important characteristic of variables and we will explore that below.
Also notice the symbol to the left of the variable name. Those symbols indicate the variable type. A numeric data looks like a box split into four quadrants. A character (i.e. text) data looks like a box split into two, with the letters 'c | h', which indicates character. Notice the size of the character variable in the image above.
Variables -- Idea 1: In MATLAB you define variables (i.e. assign data to a variable) using the equal sign (=). The variable name is on the left of the equal sign and the data is on the right. Below is an example. Run this with ctrl-enter and observe the output.
my_age = 33
The proper way to choose variable names in computing is to use an underscore to separate words and use variable names that are descriptive and intuitive.
Variables -- Idea 2: In MATLAB there are different variable sizes. Different variable sizes are used in computing because there are different types of information/data we need to store and access. The name MATLAB actually comes from one specific type of data -- a matrix. MATLAB stands for MATRIX LAB. In MATLAB each variable is actually saved as a matrix, but the size of the matrix will vary depending on the data.
We can represent almost all common data in terms of matrices. A familiar matrix that you use regularly, but that you might not associate as a matrix is a single Excel sheet. The sheet has rows (numerics: 1,2,3, etc.) and columns (letters: A,B,C, etc.).

Numerical Indexing:

In order to access data stored in a variable that is larger than size 1x1 we use indexing in MATLAB. It is similar to the idea of using the $ symbol in Excel (e.g. when we reference the information in cell $A$1). The reason we use variables is that we give a particular piece of data an intuitive name rather than use the row and column to access the information as in Excel.
In MATLAB all data is stored as a matrix. A SCALAR is a 1x1 matrix. A ROW-VECTOR is a 1xN matrix. A COLUMN-VECTOR is a Nx1 matrix. And a MATRIX can be NxN (2d) or NxNxM (3d) or NxMxKxL (4d) or any other dimension. For now we will stick to 2d matrices.
Let's investigate a 2x2 matrix and see how we retrieve individual elements of the matrix. First we need to define a variable.
my_matrix = [1 3; 2 4]
You can access and manipulate the entire matrix by using the variable name. For example:
my_matrix_plus = my_matrix + 2
Question: What does the command above do?
Answer:
You can also retrieve a single element of the matrix and operate on it. Run this for an example.
the_element = my_matrix(1,2)
We call this numerical indexing. We called row 1, column 2 and set the element of the matrix equal to a new variable called the_element.
You can also change the value of an element to a different value.
my_matrix(1,2) = 0
